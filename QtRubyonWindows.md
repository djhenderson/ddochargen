# Introduction #

This very short page details the steps necessary to develop on ddochargen under Windows.

# Details #

Following software must be installed:
  * Decent text editor. Notepad++ is preferred.
  * Mercurial client. TortoiseHG is the preferred choice.
  * Ruby 1.8.6-25
  * Qt 4.3.4
  * QtRuby

Further details can be found [here](http://vision.eng.shu.ac.uk/mmvlwiki/index.php/Qt4-QtRuby_installer_for_Microsoft_Windows).
